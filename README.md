## 📖 Sobre o Projeto

Este projeto foi criado para expandir os limites do desenvolvimento Front-End moderno. O objetivo principal é demonstrar técnicas avançadas de **CSS Grid e Flexbox**, construir um layout fiel a um design pixel-perfect do Figma e garantir uma experiência de usuário fluida em qualquer dispositivo.

### ✨ Principais Funcionalidades

* 📱 **Design Totalmente Responsivo:** Abordagem Mobile-First, adaptando-se perfeitamente de celulares a monitores ultrawide.
* 🎨 **UI/UX Moderna:** Interface limpa, baseada nos princípios atuais de design e acessibilidade.
* ⚡ **Performance:** Otimizado para carregamentos rápidos e transições de tela suaves.
* 🌙 **Suporte a Dark Mode:** Alternância dinâmica de temas para conforto visual do usuário.
* 🔄 **Integração de Dados:** Preparado para consumo de APIs RESTful.

---

## 🛠️ Stack Tecnológico

A arquitetura do projeto foi construída com as seguintes tecnologias e conceitos:

* 🧱 **[HTML5] - Estrutura Semântica:** Utilizado para criar uma base sólida, acessível para leitores de tela e otimizada para motores de busca (SEO).
* 🎨 **[CSS3] - Estilização Avançada:** Responsável por toda a parte visual, utilizando CSS Grid para layouts complexos, Flexbox para alinhamentos rápidos e `@keyframes` para animações que dão vida à interface.
* ⚡ **[JavaScript (ES6+)] - Interatividade:** Lógica de manipulação do DOM (Document Object Model), validação de formulários e controle de estados na aplicação puramente com Vanilla JS.

**🚀 O Futuro (Próximos Passos no Curso):**
* ⚛️ **[React] - Componentização:** Em breve, a interface será refatorada utilizando React para criar componentes reutilizáveis e gerenciar o estado da aplicação de forma declarativa e escalável.
* 🟢 **[Vue.js] - Reatividade:** Exploraremos também o ecossistema Vue.js, focando na sua curva de aprendizado amigável, diretivas poderosas e transições elegantes.
