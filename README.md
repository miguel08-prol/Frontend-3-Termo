## 🛠️ Tecnologias Utilizadas

O projeto foi construído utilizando as seguintes ferramentas e tecnologias:

* **[HTML5](https://developer.mozilla.org/pt-BR/docs/Web/HTML)** - Estruturação semântica.
* **[CSS3](https://developer.mozilla.org/pt-BR/docs/Web/CSS)** - Estilização, Flexbox, CSS Grid e animações.
